About This Site

This website was built using only HTML and CSS techniques covered in an introductory course: selectors, the box model (margin, padding, borders), Flexbox for layout, link states, and a simple media query for small screens.

Goals

  • Practice semantic HTML structure (header, nav, main, footer).
  • Apply CSS fundamentals for tidy, readable layouts.
  • Create a student-friendly guide to resources on implicit bias in tech.

What Changed from Project 1

  • Typography: Set a consistent font family and line height.
  • Layout: Used a max-width container for readable line lengths.
  • Navigation: Converted a plain list into a Flexbox menu.
  • Color: Added light backgrounds and hover/focus states for links.
